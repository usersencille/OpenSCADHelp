move()


use<transforms.scad>


Usage:

  • move([x], [y], [z]) ...
  • move([x, y, z]) ...


Description: Moves/translates children.

Argument

What it does

x

X axis translation.

y

Y axis translation.

z

Z axis translation.

Example 1:

#sphere(d = 10);
move([0, 20, 30]) sphere(d = 10);

Example 2:

#sphere(d = 10);
move(y = 20) sphere(d = 10);

Example 3:

#sphere(d = 10);
move(x = -10, y = -5) sphere(d = 10);
